# 🌙 Lunar Phases

## ❗ READ FIRST
This project uses an **approximate** algoritm. It does not connect to any APIs. This means that it will not be 100% accurate all the time. I've cleaned up the phase calculation a little bit but the luminocity meter is still a little funky. 
## Usage:
It's simple! Just input the date of which you'd like to view the moon's stage on!
To access it, just visit: https://jinglecats311.github.io/lunar-phases/ , or you could self-host if you're a little weird. Do whatever suits you.
